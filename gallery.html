How to Add Your Content

To add your own images and videos to this gallery:

1. Create an "images" folder and a "videos" folder in the same directory as this HTML file

2. Place your image files (JPG, PNG, GIF) in the "images" folder

3. Place your video files (MP4, WebM) in the "videos" folder

4. Replace the placeholder elements in this HTML file with actual media elements:

For images: <img src="images/your-image.jpg" alt="Description">

For videos: <video src="videos/your-video.mp4" controls></video>
